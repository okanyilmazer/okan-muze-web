# 🏛️ Okan Müze Web Arayüzü & Chatbot Entegrasyonu

Bu proje, kişisel gelişimim ve staj başvurularım için hazırladığım; modern web teknolojilerini ve 3. parti entegrasyonları nasıl kullandığımı gösteren statik bir müze web sayfası örneğidir. 

Ziyaretçilere sergiler, ajanda ve sanat eserleri hakkında bilgi sunan şık bir arayüz tasarımının yanı sıra, projede bir **AI Chatbot (Sohbet Botu)** entegrasyonu da bulunmaktadır.

## 🚀 Teknolojiler ve Araçlar

* **HTML5 / CSS3:** Sayfa iskeleti ve temel stiller.
* **Bootstrap 5:** Mobil uyumlu (responsive) tasarım, grid yapısı, carousel (kaydırıcı) ve kart bileşenleri.
* **n8n Chat Widget:** Ziyaretçilerle etkileşime girmesi için projeye entegre edilmiş yapay zeka destekli sohbet asistanı.

## ✨ Öne Çıkan Özellikler

* **Mobil Uyumlu Tasarım:** Bootstrap sayesinde sayfa, farklı ekran boyutlarına (telefon, tablet, masaüstü) otomatik olarak uyum sağlar.
* **Dinamik Görseller:** Carousel yapısı ile öne çıkan sergiler ziyaretçiye sunulmuştur.
* **Güvenlik Odaklı Geliştirme:** API ve Webhook bağlantıları güvenlik en iyi uygulamaları (best practices) gereği koddan izole edilmiş ve gizlenmiştir.
* **Chatbot Deneyimi:** Müzeyle ilgili soruları yanıtlayabilecek altyapıya sahip n8n tabanlı bir sohbet arayüzü barındırır.

## ⚙️ Kurulum ve Çalıştırma (Geliştiriciler İçin)

Projeyi kendi bilgisayarınızda çalıştırmak oldukça basittir:

1.  Bu repoyu bilgisayarınıza klonlayın:
    ```bash
    git clone [https://github.com/okanyilmazer/okan-muze.git](https://github.com/okanyilmazer/okan-muze.git)
    ```
2.  Klonladığınız dizine girin ve `index.html` dosyasını herhangi bir modern tarayıcıda (Chrome, Firefox, Safari vb.) açın.

> **⚠️ Önemli Not (Chatbot İçin):** Güvenlik nedeniyle n8n Webhook URL'si kod içerisinde `YOUR_WEBHOOK_URL_HERE` olarak değiştirilmiştir. Sohbet botunun lokalinizde çalışması için kendi n8n webhook adresinizi bu alana yapıştırmanız gerekmektedir.
